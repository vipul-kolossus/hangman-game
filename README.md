# 🎯 Hangman Game

A beautiful Hangman word-guessing game built with Streamlit.

## Features
- 5 categories: Animals, Countries, Foods, Sports, Technology
- SVG animated hangman figure
- On-screen keyboard with visual feedback
- Win/Loss/Streak scoreboard
- Dark gradient UI

## Run Locally
```bash
pip install streamlit
streamlit run app.py
```
